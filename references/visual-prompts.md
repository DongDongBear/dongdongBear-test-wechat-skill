# 视觉AI 规则

## 封面设计（3 组创意）

### Creative A: 直觉冲击型
- 用最直接的视觉隐喻表达主题
- 适合：数据类、工具类、方法论类文章
- 风格关键词：flat, minimalist, bold colors, geometric

### Creative B: 氛围渲染型
- 营造情绪和场景感
- 适合：故事类、观点类、人物类文章
- 风格关键词：soft gradient, warm light, cinematic, atmospheric

### Creative C: 信息图表型
- 用图形化方式呈现核心数据/流程
- 适合：对比类、清单类文章
- 风格关键词：infographic, clean layout, icons, data visualization

## 封面尺寸

- 微信封面比例：2.35:1
- 推荐分辨率：900×383 或 1280×544

## 内文配图规则

### 判断标准（哪些段落需要配图）
- 数据论证段：需要（用图形化呈现数据）
- 场景描述段：需要（给读者画面感）
- 转折/高潮段：需要（增强冲击力）
- 纯观点段：不需要
- 收尾/CTA段：不需要

### 约束
- 总数：3-6 张
- 间距：≥ 300 字
- 禁区：不在第一段和最后 CTA 段配图
- 尺寸：16:9 横版

## Prompt 撰写规范

1. 指定比例："16:9 aspect ratio"
2. 明确排除文字："no text, no letters, no words, no characters"
3. 指定风格：至少 3 个风格关键词
4. 留白：如果用于封面，需要 "leave space for title overlay"
5. 色调：与主题色相近的色调
6. 具体场景：不要用抽象描述，给具体的画面元素
